🧠 Engenharia de Prompt: FinChat - The Golden Standard
Este arquivo detalha a estrutura de comandos utilizada para transformar uma interface financeira comum em um Terminal de Alta Precisão (Private Banking Style) utilizando a metodologia de Vibe Coding.

🚀 O Prompt Mestre (High-Level Instruction)
Instrução: Utilize este prompt em ferramentas de geração de código (Lovable/Cursor) para replicar a inteligência e o design do sistema.

"Aja como um Engenheiro Frontend Sênior e Especialista em UX de Finanças de Alto Luxo. Refatore a aplicação seguindo estas diretrizes:

Abertura (The Golden Entry):

Implemente uma Landing Page de entrada com fundo escuro profundo e partículas sutis.

Adicione um Botão Central em degradê Dourado/Bronze Realista com efeito 'Glassmorphism'.

O dashboard deve ser revelado apenas após a interação com o botão.

Visual 'Bloomberg Intellectual':

Paleta de cores: Deep Midnight Blue (#050505), Titanium Grey e acentos Gold/Bronze.

Tipografia: Use fontes monoespaçadas (JetBrains Mono) para todos os valores numéricos para transmitir precisão técnica.

Engenharia de Precisão e Estratégia:

Implemente lógica de cálculo decimal absoluto para evitar erros de arredondamento em transações.

Adicione botões de 'Ação Rápida de Elite':

📈 Projeção de Juros Compostos: Cálculo automático de saldo futuro (12 meses a 10% a.a.).

🛡️ Score de Saúde: Algoritmo que analisa a relação Receita vs. Despesa.

🚀 Alocação Inteligente: Sugestão da regra 50/30/20 baseada no histórico de chats.

Experiência do Usuário (UX):

Chat adaptativo que reconhece entradas naturais como 'Gastei 50 com café' ou 'Recebi bônus de 2000'.

Animações de entrada em Stagger (atraso progressivo) nos cards de dashboard."

🛠️ Lógica de Construção (Arquitetura)
Contextualização de Papel: Ao definir a IA como "Especialista em Finanças de Alto Luxo", o sistema prioriza componentes de UI mais sóbrios e profissionais, fugindo do visual "app de tarefas" comum.

Restrição Numérica: A exigência de fontes monoespaçadas e precisão decimal é uma técnica de engenharia de software para garantir que a interface passe confiança ao usuário final.

Prompt de Negócio: A inclusão de estratégias como a "Regra 50/30/20" transforma um simples rastreador de gastos em uma ferramenta de consultoria financeira.
