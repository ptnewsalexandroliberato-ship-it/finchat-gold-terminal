# 🏦 FinChat: The Golden Standard Finance

[![Status](https://img.shields.io/badge/Status-Terminal_Ativo-gold?style=for-the-badge)](https://ai-purse-pal.lovable.app)

> **Engenharia Financeira de Alta Precisão.** Um terminal inteligente que processa linguagem natural para gestão de ativos e controle de fluxo de caixa com estética de luxo.

---

## 💎 Diferenciais do Terminal
* **Abertura Bronze/Gold:** Interface de entrada com autenticação visual e botão em gradiente metálico.
* **Cálculos de Precisão:** Algoritmos de processamento de transações com validação decimal absoluta.
* **Estratégias Integradas:** * Simulador de Juros Compostos (Projeção 12 meses).
    * Cálculo de Reserva de Emergência Automática.
    * Regra de Alocação 50/30/20 baseada em IA.

## 🛠️ Stack Técnica
* **Framework:** React + Vite (Modular e Eficiente)
* **Estilização:** Tailwind CSS + Framer Motion (Animações Fluídas)
* **Lógica:** TypeScript (Tipagem forte para evitar erros de cálculo)
* **Ícones:** Lucide React

---

## 🚀 Como Executar a Missão
1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Inicie o terminal: `npm run dev`
